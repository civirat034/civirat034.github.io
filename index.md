# 《零日攻擊》電視劇
《零日攻擊》（英語：Zero Day Attack）是2025年播出的台灣詩選電視劇，故事探討台灣社會在台海戰爭背景下可能面臨的狀況。該劇由高橋一生、連俞涵、杜汶澤等人擔任主演。
## 在哪裡收看？
《零日攻擊》在台灣公視主頻（CH13）与民視（CH6）首播，在LINE TV、中華電信MOD / Hami Video和台灣大哥大MyVideo等串流平台上架。
